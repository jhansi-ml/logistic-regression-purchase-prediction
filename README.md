# logistic-regression-purchase-prediction
predicting purchase based on age or salary using logistic regression
## Target
Predict whether a customer will purchase a product.

## Dataset
Purchase dataset with features:
- Age
- Estimated Salary
- Purchased

## Model
Logistic Regression

## Results
Accuracy: 0.7875

## Visualizations
- Confusion matrix
- ROC curve
- Sigmoid curve
